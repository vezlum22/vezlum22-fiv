# FIVU Sensor Firmware

This PlatformIO project targets an AVR Arduino Nano and demonstrates cooperative,
non-blocking sensor handling for:

- TCS34725 color sensor
- SHT3x temperature/humidity sensor
- MPU6050 accelerometer/gyroscope

The generated documentation is configured through `Doxyfile`.

```powershell
doxygen Doxyfile
```

Install Doxygen and Graphviz first if they are not available on the system.
